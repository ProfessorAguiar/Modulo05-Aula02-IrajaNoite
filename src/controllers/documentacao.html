Documentação da API

Rota /usuario
Rota para listar todos os usuários cadastrados

Essa rota lista todos os usuários do banco de dados com os atributos id_usuário, nome, email e senha.

Exemplo de utilização da rota /usuario:

[{"id_usuario":1,"nome":"Vinicius Aguiar","email":"vinicius@senac.com","senha":"123"},{"id_usuario":2,"nome":"joão","email":"joao@teste.com","senha":"321"},{"id_usuario":3,"nome":"admin","email":"admin@dominio.com","senha":"admin"},{"id_usuario":7,"nome":"Teste PUT","email":"alterado@","senha":"123456"}]

Rota /usuario

Rota /usuario/email/
Rota para buscar usuários por email

Essa usuários do banco de dados por email listando os atributos id_usuário, nome, email e senha. Caso não encontre o usuário com email indicado retorna uma mensagem com email não encontrado.

Exemplo de utilização da rota /usuario/email/vinicius@senac.com :

[{"id_usuario":1,"nome":"Vinicius Aguiar","email":"vinicius@senac.com","senha":"123"}]
